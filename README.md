# VGAE 
variational graph autoencoder 

## Requirements
- tensorflow                2.1.0
- spektral
- numpy
- matplotlib
- framsfiles
- fnmatch

## How To run
- unzip graphs.zip 
- open Testing_learning.ipynb and change PATH_FRAMS so it matches local dir of frams 
- run all cells (it should take max 10 min)
